Goals Page — 3 Layout Approaches

Each approach handles the same data: 1–3 active goals, each with a drill-down skill tree

A

Goal Cards (Recommended)

Each active goal is a card. Inside each card: goal name, "PRIMARY" badge, and an inline skill tree you expand to pick sub-skills. Max 3 cards. Clean separation between goals.

Pros

  • Each goal is self-contained
  • Easy to compare goals
  • Clear primary/secondary visual

Cons

  • Cards can get tall with deep trees
B

Tab per Goal

3 tabs at the top (Goal 1, Goal 2, Goal 3). Each tab shows one goal's full skill tree. The active tab is the "currently viewing" goal, not necessarily primary.

C

Split: Goals List + Skill Tree Panel

Left panel: list of 1–3 goal pills. Click one to load its skill tree in the right panel. Primary star is on the left pill.
